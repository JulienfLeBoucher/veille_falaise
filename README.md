# Projet pédagogique et associatif  pour veiller sur les falaises.

**Objectifs** :
- Créer le front-end basique d'une application web pour assurer la veille des voies d'escalade sur différentes falaises.
- Créer le back-end ad hoc pour consigner les observations faites sur le terrain (BDD, API, utilisateurs ?).
- Muter progressivement l'application web en une PWA pour permettre le relevé d'information sur le terrain offline.
